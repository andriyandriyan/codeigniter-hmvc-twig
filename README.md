# codeigniter-hmvc-twig
Boilerplate CodeIgniter using HMVC and TWIG Template

### Installation
Download or you can clone this repo using following command
```sh
$ git clone https://github.com/andriyandriyan/codeigniter-hmvc-twig.git
```
Move your directory to this folder app and then install dependency using composer
```sh
$ cd codeigniter-hmvc-twig
$ composer update
```

Run your project using following command
```sh
$ php -S localhost:3000
```
Your app is running on [http://localhost:3000](http://localhost:3000)
